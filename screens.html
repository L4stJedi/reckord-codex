⚠️ Approval notification sent by AppSheet Automation email. Manager opens AppSheet on phone or desktop, approves inline. No email reply-to-approve — must open the app. Core plan automation runs are capped (~2,000/month — sufficient for Reckord's volume).
⚠️ Multi-step wizard: AppSheet does not have a true wizard with separate pages. The "steps" are simulated using page-break sections within a single form. The step indicator is a visual-only display element — it cannot enforce sequential completion. Users could skip sections. Workaround: mark required fields in each section so the form cannot be saved incomplete.

What we CANNOT deliver with Business Starter
⚠ Limitations
🔴 External system integrations (YouTrack, etc.)
AppSheet Core cannot connect directly to REST APIs. Integration with YouTrack or other external tools requires an Apps Script middleware. Doable but adds complexity — not a simple toggle.
🔴 Fully custom-branded forms
AppSheet forms follow AppSheet's layout — you get the company colour and logo, but cannot change fonts, field layouts, or add custom CSS. Forms will look like AppSheet, not a bespoke web app.
🔴 Hard real-time conflict locking
Google Sheets has no row-level locking. Two coordinators saving at the same moment could create a duplicate booking. Warnings are shown but cannot be enforced at the database level. Acceptable for a team of this size.
🟡 PDF offer styling
AppSheet's built-in PDF is basic. A professionally branded offer PDF requires Apps Script + a Google Docs template (additional setup time: ~1 day).
🟡 Automation email volume
Apps Script: 1,500 emails/day. AppSheet Core automations: ~2,000 runs/month. For Reckord's team size this is more than enough, but worth monitoring as the team grows.
🟢 Offline mode — YES, included
AppSheet Core includes offline mode with background sync. Crew can use checklists without mobile data on-set. Requires the native AppSheet app (not browser).
🟢 Shared Drives — YES, included
Google Workspace Business Starter includes Shared Drives. All project files, offers, and attachments can live in organisation-owned folders — not tied to any individual's account.
